conectaBD()

Método que crea a conexión da base de datos.

Para realizar a conexión precisa da ruta onde está a base de datos que selle pasa no método __init__.

consultaConParametros(consultaSQL, *parametros)

Retorna unha lista cos rexistros dunha consulta realizada pasandolle os parámetros.

A consulta ten que estar definida con “?” na clausula where de SQL.

Parámetros:

  • consultaSQL – Código da consulta sql a executar

  • *parametros –

    Parámetros para introducir na consulta

Devuelve:

listaConsulta

consultaSenParametros(consultaSQL)

Retorna unha lista cos rexistros dunha consulta realizada sen pasarlle parámetros.

Parámetros:

consultaSQL – Código da consulta sql a executar

Devuelve:

listaConsulta

creaCursor()

Método que crea o cursor da base de datos.

Para realizar o cursor precísase previamente da execución do método conectaBD, que crea a conexión a base de datos na ruta onde está padada o método __init__.

pechaBD()
